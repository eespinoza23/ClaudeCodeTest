## Environment
- Python: use `py` (Windows launcher at `/c/Windows/py.exe`) — `python` and `python3` are NOT on Git Bash PATH

## Code Style
- Single self-contained HTML files (no build step, no package manager)
- Inline CSS and JS within the HTML file

## Git
- Commit after every meaningful change; push immediately after
- Commit messages: imperative, specific (e.g. `fix: clamp player to canvas bounds`)

## Communication
- Concise responses; no filler or preamble
- No emojis unless explicitly requested
